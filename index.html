Image Classification
using Tensorflow

In this project, I explored the application of various deep learning architectures to tackle the CIFAR-10 image classification challenge. Recognizing the importance of selecting the right model for the task at hand, I conducted a comparative analysis of three distinct approaches - Multi-Layer Perceptron (MLP), Convolutional Neural Network (CNN) and pre-trained ResNet50V2 model. The MLP model struggled to achieve high performance on image classication, with test accuracy of only 53%, while CNN model designed to capture spatial and hierarchical features, demonstrated significant improveemnt of 83% accuracy. The standout was ResNet50V2 model leveraging the powerful feature extraction capabilities of pre-trained network and fine-tuning it to CIFAR-10 dataset, acheiving 93% accuracy rate.

Wine Quality Prediction
using Machine Learning

In this project, I explored XGBoost classifier to predict wine quality using chemical features and categorising them into high, medium or low. Leveraging on sckit-learn library for model training, optimisation and evaluation. Sentiment Analysis of
Yelp Restaurant Reviews

In this project, I leveraged on natural language processing techniques to analyze customer sentiment towards a popular Mexican Restaurant in Singapore - Vatos Urban Tacos. By collecting and processing Yelp review data, I was able to quantify the sentiment expressed in each review, helping the Restaurant management team better understanding their customers' experiences and address any pain points. Key steps of the project: 1) Data Collection: I scraped review data from the restaurant's Yelp page, gathering 85 reivews. 2) Sentiment Scoring: Applied VADER sentiment analysis tool to each review, mapping the compound sentiment score to a 5-point scale. 3) Insights: Integrated sentiment score to a pandas dataframe, enabling exploration of trends and patterns in customer sentiment.
